# wien

簡単にミニマルデザインなテーブルグラフを作れるアプリケーションです。

## 環境構築
- nodenode@14.16.1
- npm@6.14.12
- vue-cli@4.5.12

#### 1.リポジトリをクローン
```
git clone https://github.com/uncia002/wien.git
```
#### 2.Nodeの依存関係のインストール
```
cd front
npm install
```
#### 3.開発を始める
```
npm run dev
```
